MERA 6052 is a 8080 based terminal made by CNPSS MERASTER in Katowice. The terminal uses casing from WZT TWM315 that uses casing from WZT Vela T206 B/W TV set. It was designed as a terminal to be used with MERA-60 computer system.
**HOW TO SET UP THE TERMINAL
By pressing key combination Shift+CTRL+TypeSend you enter the config menu at the bottom of the screen. Then by pressing number keys you change given parameter (Tx Rx baudrate, parity ect).
